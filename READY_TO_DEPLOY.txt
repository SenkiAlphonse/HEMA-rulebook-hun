→ Finds 3 longsword rules about strikes

"right of way target"

"penalty" with Format=COMBAT
→ Finds COMBAT format penalty rules

"equipment rapier"
→ Finds rapier-related equipment rules

## API ENDPOINTS (For Advanced Users)

Your deployed app will have:

GET /
  Returns web interface

POST /api/search
  {
    "query": "string",
    "max_results": 10,
    "formatum_filter": "VOR|COMBAT|AFTERBLOW",
    "weapon_filter": "longsword|rapier|padded_weapons"
  }

GET /api/stats
  Returns rulebook statistics

GET /api/rule/<rule_id>
  Returns specific rule (e.g., LS-VOR-1.1.1)

## HOW TO UPDATE RULES

Your rulebook is now LIVE. To update:

1. Edit markdown files (e.g., 05-hosszukard.md)
2. Optional: Update rules_index.json:
   cd qa-tools
   python parser.py
   python add_aliases.py
3. Commit and push:
   git add .
   git commit -m "Update rules"
   git push
4. Render auto-deploys (2-3 minutes)
